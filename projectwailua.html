Wailua Textiles

Problem Statement

You've come this far in HTML/CSS 101. Every site has some basic looks and function that can mocked up using just basic code skills. This is useful when you may have to reconstruct a site for a client who does not have access to their source files. Now let's try and back engineer a website that is using some advanced practices you haven't used or learned yet. Note that this will be a static landing page.

Solution

Breaking down the site from its most basic elements, I began to reconstruct the site starting with sketching out the basic portions of the site and what they do. I then decided to break out all the written text onto a basic text document. For the third task I tried to implement some of the harder pieces like a video playback and a map. My end result was pretty close aside from a few glitchy items I was fairly successful in duplicating the site.
